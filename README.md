# Agentic AI Interview Prep

This folder is a collection of agentic AI use cases built for interview preparation.

Each subfolder contains:
- A specific **use case** demonstrating a real-world agentic AI application
- The **AI agent** implementation for that use case

## Structure

```
Agentic Interview Prep/
├── ContentGeneratorForTeaching/   # AI agent that generates educational content for teaching
└── ...                            # More use cases to be added
```

## Purpose

These projects showcase how AI agents can be designed and deployed to solve practical problems — each one is self-contained and serves as a talking point for interviews around agentic AI, LLM orchestration, and multi-step reasoning systems.
